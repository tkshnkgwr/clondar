# Clondar Pro ユーザーガイド

[English Version](../en/USER_GUIDE.md) | **日本語版**

Clondar Pro をご利用いただきありがとうございます。  
本ガイドでは、アプリケーションのインストール、基本操作、時計やカレンダーのカスタマイズ、および「祝日設定マネージャー」の使用方法について詳しく説明します。

---

## 1. インストールと起動

### 1.1 動作要件
- **対応OS**: Windows 10 / 11 (64bit)
- **依存環境**: Microsoft Edge WebView2 ランタイム  
  ※ 通常の Windows 環境には標準で導入されています。未導入の場合は、起動時に自動でインストールウィザードが表示されます。

### 1.2 インストール手順
1. 配布されたインストーラーファイル（`app_x.x.x_x64_ja-JP.msi` または `app-setup.exe`）をダブルクリックして起動します。
2. インストール時の言語（日本語 / English）を選択するダイアログが表示されたら、「日本語」を選択します。
3. 画面の指示に従ってインストール先を指定し、完了ボタンを押します。
4. インストール完了後、デスクトップのショートカットまたはスタートメニューから **Clondar** を起動します。

## 2. 画面構成と基本操作

Clondar Pro は、デスクトップの壁紙に溶け込む透過型のウィジェットです。

- **ドラッグによる移動**:  
  ウィジェット内の任意の場所（時計やカレンダーの文字が無い背景部分など）をドラッグすることで、画面上の好きな位置へ自由に移動できます。
- **最前面表示（ピン留め）**:  
  カレンダーの右上にあるピンのアイコン（📌）をクリックすることで、他のアプリケーションを最大化していても、常にウィジェットを一番手前に表示し続ける（Always on Top）ことができます。もう一度クリックすると解除されます。
- **表示・非表示の切替**:  
  ウィジェット右上の「❌」ボタンをクリックするか、キーボードの `Esc` キーを押すと画面が消えます。これは一時的に非表示（常駐状態）にしているだけで、後述するシステムトレイからいつでも再表示できます。

## 3. 時計・カレンダー機能の使い方

### 3.1 時計モードの切替
時計表示の右側にあるトグルスイッチ（アナログ／デジタルのアイコンなど）をクリックすることで、表示形式を変更できます。
- **デジタル時計**: `Impact` 調のシャープで力強いフォントで時刻を表示します。
  - **12H/24H 切替**: 時刻表示部分をダブルクリック、または設定コントローラーから12時間表示と24時間表示を切り替えることができます。
  - **秒表示のON/OFF**: 必要に応じて秒表示のサブ時計を非表示にし、より静的な表示にカスタマイズできます。
- **アナログ時計**: 針が滑らかに動くミニマルな時計です。

### 3.2 カレンダー表示
- **月間表示**: 常に6週間分（42日間）の枠組みで表示され、曜日の配置やカレンダーの高さが毎月変動しないため、視覚的なブレがありません。祝日は赤色で表示され、マウスをホバーすると祝日名がツールチップで表示されます。
- **年間表示**: 年月ヘッダーの横にある「カレンダー」アイコンをクリックすると、画面いっぱいに当年の年間カレンダーが展開されます。前後の年への切り替えにも対応しています。

## 4. 祝日設定マネージャーの使い方

カレンダーの右上のギアアイコン（⚙️）または「祝日設定」ボタンをクリックすると、日本の祝日を独自にカスタマイズ・編集できる「祝日設定マネージャー」画面が開きます。

### 4.1 祝日のビジュアル編集
- **祝日の追加**:  
  画面左下の入力フォームより、「日付（月・日）」と「祝日名」を入力して「追加」ボタンを押すと、リストにカスタム祝日が一時追加されます。
- **祝日の削除**:  
  祝日一覧リストの各行の右側にある「ゴミ箱」アイコンをクリックすることで、不要な祝日を一覧から削除できます。

### 4.2 差分表示 (デフォルト比較)
画面中央には、アプリに組み込まれている「デフォルトの祝日設定」と、あなたがカスタマイズした「現在の設定」との行単位の差分（LCSアルゴリズムによる計算）がカラーハイライトで視覚的に表示されます。
- **緑色のハイライト（+）**: あなたが新しく追加した祝日です。
- **赤色のハイライト（-）**: デフォルトから削除された祝日です。

### 4.3 保存と適用
編集した祝日データを反映させるには、画面右下の「**保存して適用**」ボタンを押します。
- データは OS のユーザーローカルデータ領域（LocalAppData）へ `%LOCALAPPDATA%/com.clondar.pro/holidays.json` として安全に永続化保存されます。
- 保存した内容はカレンダー画面に即時リロードされ、再起動後も自動で維持されます。

## 5. システムトレイメニューの利用

Clondar Pro は、起動するとタスクバーの右端（システムトレイ）に常駐します。トレイ内の Clondar アイコンを**右クリック**すると、以下の機能にアクセスできます。

- **表示 / 非表示**: 非表示にしたウィジェットを再表示したり、逆に画面から一時的に消したりします。
- **最前面表示の切替**: カレンダー画面のピンアイコンと同様に、常に一番手前に表示するかどうかを切り替えます（トレイメニュー側でチェックマークが連動します）。
- **位置をリセット**: ディスプレイの接続変更や解像度の変更によってウィジェットが画面外に隠れて見えなくなってしまった場合、この項目を選択すると、ウィジェットが即座にメインディスプレイの中央に引き戻されます。
- **終了**: アプリケーションを完全に終了し、常駐を解除します。
